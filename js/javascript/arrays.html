Arrays

It is a kind of data Structure, Where  we can store multiple values
In Js arrays are dynamic in nature
Arrays methods -> push() -> to insert an element at the end of an Array
                           pop() -> to delete an element at the last index of an Array
                           unshift() -> push the data at the beginning of an Array
                           shift() -> remove an element at the beginning of an Array
                           indexOf() -> specific element based on index
                           join() -> join all the elements in an array and return it in string format
                           includes() -> check whether their or not 
                           reverse() -> reverse the array 
                           slice() -> same as string splice
